Como ler as três lentesHow to read the three lenses
  • Ganho vs. plano (No-AI ÷ AI plan) — o número conservador que publicamos (2,5×). Não depende de ninguém registrar o real. É a estimativa planejada do ganho vs. um sênior sem IA.Gain vs plan (No-AI ÷ AI plan) — the conservative number we publish (2.5×). Doesn't depend on anyone logging actuals. It's the planned gain vs. a senior without AI.
  • Ganho vs. real (No-AI ÷ AI real) — o ganho realizado, usando o tempo real (5,2×). Maior porque o agente superou o plano. Só é confiável na medida em que o CompletedWork é registrado honestamente.Gain vs real (No-AI ÷ AI real) — the realized gain, using logged actuals (5.2×). Bigger because the agent beat its plan. Only as trustworthy as the honestly-logged actuals.
  • Precisão da estimativa (AI plan ÷ AI real) — não é um ganho: mede o quão perto a estimativa agentic ficou do real. 1,0× = acertou em cheio (todas as tasks de BE); maior = superestimamos. Útil para calibrar planejamento.Est. accuracy (AI plan ÷ AI real) — not a gain: how close the agentic estimate was to reality. 1.0× = nailed it (all BE tasks); higher = we over-estimated. Useful to calibrate planning.
  • Estimativa manual (No-AI) = dev sênior fazendo a task sem IA, ciclo completo (entender → implementar → testar → PR → review). Estimada task a task pelo escopo real entregue, não por fórmula. Multiplicadores deliberadamente conservadores.Manual estimate (No-AI) = a senior dev doing the task without AI, full cycle (understand → implement → test → PR → review). Estimated task by task from the real scope delivered, not a formula. Deliberately conservative multipliers.
